AJAX — это аббревиатура, которая означает Asynchronous Javascript and XML.
AJAX это способ доступа к веб серверам с веб страницы.

при использовании AJAX нет необходимости обновлять каждый раз всю страницу,
так как обновляется только ее конкретная часть.

Обобщим достоинства AJAX:
Возможность создания удобного Web-интерфейса
Активное взаимодействие с пользователем
Частичная перезагрузка страницы, вместо полной
Удобство использования

Все современные браузеры (Chrome, Firefox, IE7+, Edge, Safari, Opera) имеют встроенный XMLHttpRequest объект.

Чтобы отправить запрос на сервер, мы используем методы open() и send() объекта XMLHttpRequest:

xhttp.open("GET", "ajax_info.txt", true);
xhttp.send();

open(1method, 2url, 3async)
method: тип запроса: GET или POST
url: расположение сервера (файла)
async: true (асинхронный) или false (синхронный)
Указывает тип запроса

send()  -  Get
send(string)  -  Post
Отправляет запрос на сервер

Get проще и быстрее чем Post, но по сравнению с Get он более надежен и безопаснее, у него нет ограничении по размеру.

 Get запрос
xhttp.open("GET", "demo_get.asp", true);
xhttp.send();

POST запрос
xhttp.open("POST", "demo_post.asp", true);
xhttp.send();
